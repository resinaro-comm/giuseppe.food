# Videos Directory

Place your downloaded Instagram videos here with names matching the recipe slugs:

- `roast-chicken.mp4`
- `beef-braciole.mp4`
- etc.

See `VIDEO_SETUP.md` in the root directory for download instructions.

## Quick Download Example

```bash
# Using yt-dlp
yt-dlp "https://www.instagram.com/reel/DRJ0tEPAqba/" -o "roast-chicken.mp4"
```
